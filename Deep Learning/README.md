## Pengenalan

Deep Learning adalah cabang dari pembelajaran mesin yang menggunakan jaringan saraf tiruan berlapis (deep neural networks) untuk mempelajari representasi fitur bertingkat dari data. Model deep learning mampu menangkap pola kompleks pada data besar dan sering kali outperform metode tradisional pada tugas seperti penglihatan komputer, pemrosesan bahasa alami, dan pengenalan suara.
